# e-roux-plugins — Unified General-Purpose Development Environment

This suite registers global development guards, Ansible Molecule workflow helpers, and Make-centric workflow enforcers.

- **dev**: General development guards (blocking credentials, direct main pushes, bad migrations).
- **make**: Make-centric workflow enforcer.
- **infra**: Molecule + Podman driver automation.
- **vulcan**: Vulcan development guidelines and workspace patterns.
